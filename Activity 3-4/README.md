# Coffee Shop Rewards System

This is a simple Java program that simulates a coffee shop reward system.
It shows how customers earn points and redeem rewards.

---

## Purpose
This project is made to practice basic Java concepts such as:
- Classes and objects
- Constructors
- Instance and static variables
- Methods and static methods
- Simple conditions

---

## What the program does
- Creates customer objects
- Adds reward points to customers
- Allows customers to redeem rewards
- Displays customer information
- Counts total registered customers
- Shows how many rewards a customer can redeem

---

## Program Limitations
- Console-based only
- No user input
- Data is not saved after the program ends
