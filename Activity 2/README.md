# My Book List (Java)

This program is a simple Java application that shows how to create and use a `Book` class.
It prints book details and demonstrates changing values using methods.

---

## Purpose
This project is made to practice basic Java concepts such as:
- Classes and objects
- Constructors
- Methods
- Arrays
- Simple conditions

---

## What the program does
- Creates book objects
- Displays book details
- Changes the current page of some books
- Changes the author of a book
- Displays books published after 2010

---

## Program Limitations
- Data is not saved after the program ends
- Console output only
- No user input
