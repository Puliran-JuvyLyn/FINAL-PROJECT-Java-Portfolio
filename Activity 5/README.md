# Teacher and Student Assignment (Java)

This is a simple Java program that shows the relationship between
Student, Teacher, and Course using classes and inheritance.

---

## Project Description
The program creates:
- One student
- One teacher
- Two courses

It then displays the information of the student, teacher, and course
using basic Java methods.

## What the Program Does
- Creates a Student object
- Creates Course objects
- Assigns courses to a Teacher
- Displays student information
- Displays teacher information
- Displays course information

---

## Program Limitations
- No user input
